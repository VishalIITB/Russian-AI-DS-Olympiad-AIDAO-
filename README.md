# Quantum-Inspired Time Series Forecasting

## 🚀 Overview
This project implements a multivariate time-series forecasting pipeline using quantum-inspired feature engineering.

## ⚛️ Key Idea
We map classical features into a higher-dimensional space using sine/cosine transformations inspired by quantum state encoding.

## 📊 Dataset
Jena Climate Dataset (2009–2016)

## 🧠 Models Used
- LSTM
- Transformer

## 📉 Metrics
- MAPE (Mean Absolute Percentage Error)

## 🛠️ Tech Stack
- Python
- PyTorch
- NumPy, Pandas

## 📌 Results
Quantum feature encoding improves model performance by capturing nonlinear dependencies.
